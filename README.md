# Pokémon Harp and Guitar - Releases

Welcome to the official public release page for **Pokémon Harp and Guitar**.

This repository is used only to host playable demo builds of the game.  
The main development repository is private.

## Downloads

You can download the latest available version from the **Releases** section.

## Current Versions

- **v0.1** - Playable up to the first Gym
- **v0.2** - Playable up to the second Gym

## Important Notice

This game is still in development.  
Bugs, unfinished areas, balance issues, and missing content may be present in the current demo versions.

## Future Updates

More content, features, fixes, and improvements will be added in future versions.
Version **v0.3** is planned to include a larger amount of new implemented content.

Thank you for playing and supporting the project.
